# Olist Business KPI Dashboard

A Business KPI Dashboard built using Power BI to analyze e-commerce performance and derive actionable insights.

## Overview
This project analyzes the Olist e-commerce dataset to track key business metrics such as revenue, orders, customers, and delivery performance. The dashboard provides a clear view of trends, top-performing products, and customer behavior.

## Key KPIs
- Total Revenue: 16.01M  
- Total Orders: 99K  
- Total Customers: 96K  
- Average Order Value: 160.99  
- Average Delivery Time: 12 days  

## Dashboard Insights
- Revenue trend over time  
- Top products by revenue  
- Revenue by product category  
- Revenue by payment method  
- Top customers by revenue  
- Delivery time distribution
